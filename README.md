# Svelte with spa router

Example using [svelte-spa-router](https://github.com/ItalyPaleAle/svelte-spa-router) with nested routes and context.

---

Install dependencies
```
npm i
```

Development mode
```
npm run dev
```

Build and run server
```
npm run build
npm start
```
